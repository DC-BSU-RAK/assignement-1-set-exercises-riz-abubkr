HTML & CSS Quick Reference Guide

HTML Tags

<h1> to <h6>

Description: Heading tags are used to define headings of various levels (h1 being the largest).

<h1>This is a level 1 heading</h1>

<p>

Description: Paragraph tag defines a block of text.

<p>This is a paragraph of text.</p>

<a>

Description: Anchor tag Help to define hyperlinks.

<a href="https://example.com">Click here</a>

<ul> and <ol>

Description: The unordered list (<ul>) and ordered list (<ol>) tags define lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<div>

Description: Div tag is a block-level container for grouping content.

<div>This is a div container</div>

<img>

Description: Image tag Help to add images in a webpage.

<img src="image.jpg" alt="Description">

<form>

Description: The form tag is used to collect user input.

<form action="submit_form.php">
  <input type="text" name="username">
  <button type="submit">Submit</button>
</form>

<span>

Description: The span tag is an inline container for text or elements.

<span>This is inline text</span>

<strong>

Description: The strong tag is used to define important or bold text.

<strong>This text is bold and important</strong>

<table>

Description: The table tag defines a table in HTML.

<table>
  <tr><th>Header 1</th><th>Header 2</th></tr>
  <tr><td>Data 1</td><td>Data 2</td></tr>
</table>

CSS Properties

color

Description: The color property sets the color of text.

p {
    color: red;
}

background-color

Description: The background-color property sets the background color of an element.

div {
    background-color: lightgray;
}

font-size

Description: The font-size property sets the size of the text.

h1 {
    font-size: 32px;
}

margin

Description: The margin property defines the space around elements.

div {
    margin: 20px;
}

padding

Description: The padding property defines the space between an element's content and its border.

div {
    padding: 15px;
}

border

Description: The border property sets the border of an element.

div {
    border: 1px solid black;
}

display

Description: The display property specifies how an element is displayed on the webpage.

div {
    display: inline-block;
}

text-align

Description: The text-align property sets the horizontal alignment of text.

h1 {
    text-align: center;
}

font-family

Description: The font-family property defines the font of an element.

p {
    font-family: Arial, sans-serif;
}
